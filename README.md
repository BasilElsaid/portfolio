# Portfolio Website

Portfolio personale sviluppato con Angular per presentare i miei progetti, competenze ed esperienza come sviluppatore software.

## Sito Live

👉 [Visualizza il sito](https://portfolio-basil-elsiad.vercel.app)

---

## Descrizione

Questo è il mio sito portfolio personale, dove raccolgo:

- Il mio percorso accademico in Informatica
- Le competenze nello sviluppo backend e frontend
- I principali progetti personali e universitari
- I miei contatti e link social

L’obiettivo è avere una presenza web moderna, veloce e minimal che rappresenti il mio lavoro come sviluppatore.

---

## Tecnologie utilizzate

- Angular
- TypeScript
- HTML / CSS
- Tailwind CSS
- Vercel (deploy)

---

## Funzionalità

- Design responsive (desktop e mobile)
- Interfaccia minimal e moderna
- Sezione progetti
- Sezione "About me"
- Download CV
- Sezione contatti con link social
- Routing SPA lato client
